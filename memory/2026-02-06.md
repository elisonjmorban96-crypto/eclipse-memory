### Database Schema

**4 tables:**
1. `jobs` — Core Upwork job data + submission history
2. `job_status_log` — Audit trail (drafted → submitted → viewed → replied)
3. `runs` — Track each automation run (jobs found, submitted, errors)
4. `follow_ups` — Track follow-up messages sent to clients

### Features Locked In

**Filtering rules** (from config.json):
- Keywords: "AI automation", "Zapier", "CRM", "lead gen", "chatbot", etc.
- Blocklist: NFT, crypto, betting, adult, unpaid test, commission-only
- Thresholds: $30+/hr, $200+ fixed, client rating ≥4.5
- Data captured: job title, client, budget, posted time, proposal text, status timeline

**Operational flows:**
- Search → Score → Draft → Approval Gate → Submit → Follow-up
- Status pipeline: shortlisted → drafting → awaiting_approval → submitted → viewed → replied → interview → hired/rejected
- Follow-ups: scheduled at 3, 7, 14 days post-submission
