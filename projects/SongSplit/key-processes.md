# SongSplit — Key Processes

## User Journey

### 1. Signup / Onboarding
- Create account (email or Google)
- Quick tutorial: "How splits work"
- Prompt to start first project

### 2. Project Creation
- Name the song/project
- Invite collaborators (email or link)
- Define contribution categories (lyrics, production, mixing, etc.)

### 3. Split Agreement
- Default split: equal among contributors
- Adjust percentages based on contributions
- All parties digitally sign
- Agreement timestamped and stored

### 4. Contribution Tracking
- Log work sessions
- Attach files/versions
- Notes about specific contributions

### 5. Export & Legal
- Generate PDF agreement
- Export split sheet for distributors
- Optional: connect lawyer for review

### 6. Release & Royalties
- Tag project as "released"
- Input royalty amounts (manual until integrations)
- Calculate and display individual shares
- Payout tracking (manual until integrations)

## Admin Processes

### Beta Management
- Track beta users in GHL
- Weekly check-in emails
- Feedback collection via Typeform/Notion
- Feature prioritization based on feedback

### Content Creation
- Blog: "Understanding Music Splits"
- Social: Artist success stories
