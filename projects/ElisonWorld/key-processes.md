# ElisonWorld — Key Processes

## Content & Release Management

### Existing Catalog
- Ensure all releases are live on major platforms
- Monitor for takedowns or distributor issues
- Update bios and photos annually

### New Releases (if activated)
1. Pre-release: 4-week runway
2. Distribution setup (DistroKid / TuneCore)
3. Marketing assets from OneTime Studios
4. Social media campaign
5. Release day coordination
6. Post-release: 2-week engagement push

## Social Media
- Posting schedule: 1x/week minimum
- Content types: Behind-the-scenes, studio clips, throwbacks, SongSplit features
- Platforms: Instagram primary, TikTok secondary

## Royalty Tracking
- Quarterly review of distributor statements
- Verify payments match streaming data
- Document for tax purposes

## Collaboration Inquiries
- Screen for brand alignment
- Check calendar availability
- Evaluate financial/terms
- Respond within 48 hours (yes/no/maybe)

## Performance Opportunities
- Evaluate based on:

  - Audience fit
  - Travel/logistics
  - Brand alignment
- Minimum 2-week lead time preferred
